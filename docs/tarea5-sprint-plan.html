1. Ficha del Proyecto

NombreBrújula Futura
TipoAplicación Web de Orientación Vocacional
Público ObjetivoEstudiantes de bachillerato (15-18 años), especialmente de colegios Fe y Alegría
ProblemaLos estudiantes no tienen claridad sobre qué estudiar porque la orientación que reciben es muy general e insuficiente
SoluciónExperiencia interactiva, visual y guiada para explorar carreras, comparar opciones y encontrar un camino claro
StackReact 19 + Vite 8 · Vanilla CSS · Deploy en Vercel
Repositoriogithub.com/floressemily/Br-jula-Futura
URL DeployPendiente de configurar en Vercel

4. Definition of Done (DoD)

Una funcionalidad se considera terminada cuando cumple todos estos criterios:

  • ✅ Está implementada en código y funciona correctamente
  • ✅ Ha sido probada manualmente en Chrome y en dispositivos móviles
  • ✅ El código fue revisado mediante Pull Request o revisión de pares
  • ✅ Fue subida al repositorio en la rama correspondiente
  • ✅ Está desplegada y visible en la URL de producción (Vercel)
  • ✅ No presenta errores en consola del navegador
  • ✅ Es responsive (funciona en escritorio y móvil)
